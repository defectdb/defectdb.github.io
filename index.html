DefectDB¶

Welcome to DefectDB!¶

This project aims to be a source of detailed and well-structurized information about program defects.

Integration¶

We believe that analyzers and compilers emitting defect reports might benefit from it, so feel free to integrate this database to your project.

Contributing¶

Feel free to contribute to this project by submitting pull requests.
